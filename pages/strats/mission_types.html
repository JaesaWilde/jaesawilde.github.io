Mission types

The first crucial step is to understand the four following mission types:

  • Burn em all mission
  • Run and burn mission
  • One and out missions
  • Fullpod mission

These missions involve a main objective requiring completion times which cannot be shortened.

These moments are long enough to allow you to burn all reinforcements all along and kill yourself for the last time as soon as the Mission completed message appears on screen.

It is therefore mandatory to spot every and each of those incompressible moments in order to properly burn reinforcements as it shall not slow down main objective completion.

Each Burn em all mission has a dedicated page to be found on the left bar menu. They contain detailed explanations about death optimization.

Run and burn

These missions allow you to burn part of the reinforcements while main objective is being completed. You will however get it done before having zero reinforcements left.

A precise strategy is then required to burn as many reinforcements as you can without slowing down main objective completion. Left reinforcements will be burned in full squad death rounds.

Each Run and burn mission has a dedicated page to be found on the left bar menu. They contain detailed explanations about death optimization.

These missions are amid the fastest ones to complete.

They do not belong to the Fullpod mission category because you will need to call in a single stratagem to get main objective completed before going through the six rounds of death.

Unlike Run and burn missions, you won't have time to burn reinforcements while completing main objective.

Each One and out mission has a dedicated page to be found on the left bar menu. They contain detailed explanations about death optimization.
